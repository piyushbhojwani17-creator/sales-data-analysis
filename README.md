# 📊 Sales Data Analysis Project

## 📌 Objective

Analyze retail sales data to identify trends, top-performing products, and regional performance.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Key Analysis

* Sales by Category
* Profit by Region
* Top Products
* Monthly Sales Trend

## 🔍 Insights

* Technology category generates highest sales
* West region gives highest profit
* Top products contribute major revenue
* Sales show seasonal trends
* Profit is not always proportional to sales

## 📁 Files

* sales_analysis.ipynb → Main project notebook
* Sample - Superstore.csv → Dataset
